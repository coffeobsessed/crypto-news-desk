# Как опубликовать генератор

Самый простой вариант для теста с подругой — Render.

## 1. Загрузить проект на GitHub

1. Создайте новый репозиторий на https://github.com/new
2. Название, например: `crypto-news-desk`
3. Загрузите в него файлы из этой папки.

## 2. Опубликовать на Render

1. Откройте https://render.com/
2. Нажмите `New +`
3. Выберите `Web Service`
4. Подключите GitHub-репозиторий с проектом
5. В настройках укажите:

- Runtime: `Node`
- Build Command: оставить пустым или `npm install`
- Start Command: `npm start`

Render сам даст публичную ссылку. Ее можно отправить подруге.

## Важно

Это приложение ходит за новостями в интернет, поэтому его нельзя нормально опубликовать как простой статический сайт. Нужен сервис, который умеет запускать Node-сервер.
